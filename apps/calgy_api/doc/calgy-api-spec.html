post /calendars

Create a new calendar.

  • Request
  • Response

Body

Media type: application/json

Type: object

Example:

{}

HTTP status code 201

Headers

  • location: required (string)

    Location of the created calendar resource.

    Example:

    http://api.calgy.org/calendars/413cd8cf-71f0-47a0-8ce4-644a95dba69e;admin

Body

Media type: application/json

Type: object

Properties
  • id: required (string - pattern: ^[0-9a-f]{8}(?:-[0-9a-f]{4}){3}-[0-9a-f]{12}$)

    Example:

    b376e8ff-b222-4559-a0e7-f344e505cee8
  • state: required (pending)

    Example:

    pending
  • title: (string - maxLength: 100)

    Example:

    Your Calendar
  • description: (string - maxLength: 2000)

    Example:

    The calendar that started evolving.
  • public_url: required (string)

    Example:

    http://api.calgy.org/calendars/b376e8ff-b222-4559-a0e7-f344e505cee8
